03

Voice & Copy

✓ Do
Lead with outcomes (pipeline, revenue, citations) — not features.
Use specific numbers. "384% organic traffic growth" not "significant growth."
Short sentences. Strong verbs. Active voice always.
Sentence case headlines: "GEO is the new SEO."
Direct CTAs: "Book an intro call" / "Get the audit" / "See it live."
Name the pain before offering the solution.
✗ Don't
Don't open with "We are excited to announce..."
Don't use passive voice. "Revenue was increased" → "We increased revenue."
Don't pad with qualifiers: "very", "quite", "somewhat."
Don't say "Click here to learn more about our platform."
No em dashes (—). Use en dashes (–) for pauses and ranges.
Don't use jargon to sound smart. Only use it when it's precise.

# RevenueZen Brand System

## 1. Colors
| Token | Hex | Use |
|---|---|---|
| Near Black | #0a0f0d | Primary text, nav bg |
| Green Deep | #002910 | Hero backgrounds, dark sections |
| Green Brand | #00a846 | Primary CTAs, links |
| Accent | #00e676 | Hover states, highlights on dark |
| Green Light | #ccffe0 | Pill fills, tinted surfaces |
| Yellow | #d4ff6e | Label pills (DM Mono only) |
| Text Secondary | #5a7260 | Captions, supporting copy |
| Stroke | #e8f0eb | Borders, dividers |

## 2. Typography
| Style | Font | Size | Weight |
|---|---|---|---|
| Display/H1 | DM Serif Display | 64-96px | 400 |
| H2 | DM Serif Display | 40-56px | 400 |
| H3 | DM Sans | 24-28px | 600 |
| Body | DM Sans | 16-18px | 400 |
| Label/Pill | DM Mono | 11-12px | 500 |

Rules: No border-radius anywhere (sharp corners). Sentence case headlines. Tracking -0.02em on display. DM Mono labels always uppercase, letter-spacing 0.08-0.12em.

## 3. Voice & Copy
Personality: Direct. Results-focused. Expert but human. Active voice always.

DO:
- Lead with outcomes (pipeline, revenue, citations) not features
- Use specific numbers: "384% traffic growth" not "significant growth"
- Short sentences. Strong verbs. No padding.
- Sentence case headlines
- CTAs: "Book an intro call" / "Get the audit" / "See it live"
- Name the pain before offering the solution

DON'T:
- Open with "We are excited to announce..."
- Use passive voice
- Use em dashes -- use en dashes (-) for pauses and ranges
- Say "Click here to learn more"
- Use qualifiers: very, quite, somewhat

BANNED: synergize, robust, comprehensive, seamless, leverage (verb), supercharge, groundbreaking, revolutionary, game-changing, holistic, best-in-class, cutting-edge

POSITIONING:
- GEO (never "AI SEO" or "LLM SEO")
- Pipeline as primary outcome (not just "leads" or "traffic")
- SME-led for expert content (not "AI content")

## 4. Components
Buttons: No border-radius. Primary (#00a846 bg, white text). Secondary (outline, #002910). Ghost (white outline for dark). DM Sans 600 15px. Padding: 14px 28px.

Pills: No border-radius. DM Mono, 11px, 500, uppercase. Yellow (#d4ff6e) featured. Green (#ccffe0) categories.

Cards: White bg. 1px #e8f0eb border. No radius. Padding 28px. DM Mono labels, DM Serif Display titles, DM Sans body.

## 5. How to Apply
Copy: Direct, outcome-led, active voice. No banned words. Sentence case. GEO/Pipeline/SME-led positioning.
HTML/CSS: Sharp corners everywhere. Green palette. DM Serif Display headlines. DM Sans body. DM Mono labels uppercase.
Data viz: Green spectrum. Sharp corners. DM Serif Display headlines. DM Mono axis labels. No gradients.
